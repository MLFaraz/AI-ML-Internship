# Day 3 - OOP and File Handling

## Practical Tasks

1. student.py - Student class with name, roll, dept
2. employee.py - Employee class with salary calculation
3. inheritance.py - Person, Student, Teacher classes
4. polymorphism.py - Different classes with same method
5. read_file.py - Read and display text file
6. student_records.py - Add and view student records in file
7. error_handling.py - Try and except for file errors

## Projects

### Library Management System (library.py)
- Add a Book
- View All Books
- Search by Title
- Issue a Book
- Return a Book
- Saves data in books.txt

### Intern Management System (intern.py)
- Add a New Intern
- View All Interns
- Search by ID or Name
- Assign a Daily Task
- Update Task Status
- Delete an Intern Record
- Saves data in interns.txt

## How to Run
```
python student.py
python library.py
python intern.py
```

## What I Learned
- Classes and objects
- Inheritance and polymorphism
- Reading and writing files
- Using try and except
- Building menu-based programs

## Challenges
- Saving and loading data from text files
- Handling invalid user input
- Keeping data after program closes

## How I Solved Them
- Used comma-separated format in text files
- Added input checks before saving
- Loaded data when program starts and saved on every change
